## Getting Started

### Installation

With [Cargo](https://doc.rust-lang.org/cargo/) installed, build and install from latest main with
```bash
cargo install --git https://github.com/lunabunn/cubesteak
```

## Usage

Simply run:
```bash
cubesteak <source_dir> <target_dir>
```
